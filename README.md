# 🛒 Telegram Reseller Bot

Бот для перепродажи товаров с автоматическим парсингом источника, наценкой по категориям и публикацией в ваш канал.

---

## ⚙️ Возможности

| Функция | Описание |
|---------|----------|
| 🔄 Автопарсинг | Читает товары из канала-источника через Telethon |
| 🏷️ Наценки | Разные % для каждой категории товаров |
| 📢 Автопостинг | Публикует товары в ваш канал по расписанию |
| 🔍 Поиск | Клиенты ищут товары прямо в боте |
| 💬 Заявки | Все запросы на заказ приходят вам в личку |
| 📊 Статистика | Пользователи, заказы, сообщения |
| 📣 Рассылка | Массовые сообщения всем пользователям |

## 🚀 Быстрый старт

### 1. Установка зависимостей

```bash
pip install -r requirements.txt
```

### 2. Настройка

```bash
cp .env.example .env
nano .env   # Заполните своими данными
```

**Что нужно заполнить в .env:**

- `BOT_TOKEN` — получить у [@BotFather](https://t.me/BotFather)
- `ADMIN_IDS` — ваш Telegram ID (узнать у [@userinfobot](https://t.me/userinfobot))
- `MY_CHANNEL_ID` — юзернейм вашего канала (бот должен быть там **администратором**)
- `SOURCE_CHANNEL_ID` — канал откуда брать товары
- `TELEGRAM_API_ID` + `TELEGRAM_API_HASH` — с [my.telegram.org/apps](https://my.telegram.org/apps)

### 3. Настройка наценок

В `config.py` найдите `MARKUP_RULES` и укажите свои категории:

```python
MARKUP_RULES = {
    "электроника": 20,   # +20% к цене
    "одежда": 30,        # +30% к цене
    "обувь": 25,
    "default": 15        # для остальных категорий
}
```

### 4. Запуск

```bash
python bot.py
```

## 🔐 Админ-панель `/admin`

После ввода `/admin` открывается меню с кнопками:

- **➕ Добавить товар** — ручное добавление: название, цена, категория, описание, фото
- **📊 Статистика** — пользователи, товары, заказы
- **💬 Последние сообщения** — все переписки клиентов с ботом
- **🛒 Заказы** — список всех запросов на заказ
- **🏷️ Изменить наценки** — поменять % для любой категории
- **📢 Опубликовать все товары** — разослать все товары в канал
- **📣 Рассылка** — сообщение всем пользователям бота

## 🤖 Как работает парсинг

### Автоматический (через Telethon)

Бот читает сообщения из канала-источника и ищет:
- **Название** — первая строка сообщения
- **Цену** — по ключевым словам: "цена", "₽", "руб", "💰"
- **Категорию** — по ключевому слову "категория:" или хэштегам `#электроника`

**Формат сообщений в канале-источнике:**
```
Название товара

Описание товара (необязательно)

Категория: Электроника
Цена: 5000 руб
```

### Ручное добавление

Через `/admin` → "Добавить товар" — введите данные вручную.

## 🖥️ Запуск на сервере (systemd)

```ini
# /etc/systemd/system/reseller-bot.service
[Unit]
Description=Telegram Reseller Bot
After=network.target

[Service]
Type=simple
User=ubuntu
WorkingDirectory=/home/ubuntu/telegram_reseller_bot
EnvironmentFile=/home/ubuntu/telegram_reseller_bot/.env
ExecStart=/usr/bin/python3 bot.py
Restart=always
RestartSec=10

[Install]
WantedBy=multi-user.target
```

```bash
sudo systemctl enable reseller-bot
sudo systemctl start reseller-bot
sudo systemctl status reseller-bot
```

## ❓ Частые вопросы

**Q: Бот не читает канал-источник**  
A: Убедитесь что заполнены `TELEGRAM_API_ID` и `TELEGRAM_API_HASH`, и вы авторизованы через Telethon (при первом запуске введёт номер телефона).

**Q: Бот не постит в мой канал**  
A: Добавьте бота в канал как **администратора** с правом публикации сообщений.

**Q: Как добавить новую категорию?**  
A: Добавьте в `MARKUP_RULES` в `config.py` или используйте `/admin` → "Добавить товар" с новым названием категории — она создастся автоматически.
